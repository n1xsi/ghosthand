# ghosthand

**GhostHand** – это автоматизированный софт с набором скриптов для помощи в FPS-играх.

Проект является чисто _external assistaint tool_ для небольшой помощи в игре.

> [!Important]
> Repo still in development. **Everything is in its "raw" form.**

> [!WARNING]
> Данный софт – это заскриптованная имитация нажатий клавишей клавиатуры/мыши, поэтому какие-то функции могут работать дёргано, некорректно или вовсе не работать (в зависимости от игры).
> 
> Для лучшей работы программы, её необходимо запускать от имени администратора, а в играх использовать режим отображения окна "Оконный" или "Полноэкранный в окне".
> 
> Все риски <b><i>Вы берёте на себя</i></b>, ибо за подозрительные действия в играх вас могут забанить. 

## ⚙ Стек:
* Python 
* DearPyGui
* ctypes
* threading
